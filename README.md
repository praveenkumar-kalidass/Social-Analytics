# Webpack-React-Grunt
A UI Showcase

## Specifications
 - Webpack 4.x
 - React 16.x.x
 - Bootstrap

# Requirements
 - Node.js - latest v8.x

## Installation
 - `npm install` - Node modules

## Usage
 - `npm run grunt-dev` - Run with Webpack dev server
 - `npm run grunt-build` - Build production-ready code
